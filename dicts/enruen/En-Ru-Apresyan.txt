В состав данного архива входят исходник самого словаря ("En-Ru-Apresyan.dsl") и исходник словаря аббревиатур ("En-Ru-Apresyan_abrv.dsl"). Необходимо скомпилировать оба этих словаря.

Для компиляции основного словаря и словаря аббревиатур необходимо указать следующие параметры:

1. Указать кодировку исходного файла, как "Unicode".
2. Убрать галочку "Автоматически добавлять разметку, необходимую для поиска по содержимому карточек словаря".

Чтобы начать работу со словарем необходимо:

1. Подключить скомпилированный основной словарь ("En-Ru-Apresyan.lsd") к Lingvo.
2. Вызвать окно с настройками Lingvo и указать Юникод шрифт для отображения текста карточек. На настольном компьютере в качестве такого шрифта можно использовать 'Arial Unicode MS', который входит в состав Microsoft Office. На КПК можно использовать шрифт 'DejaVu Sans', который можно бесплатно скачать с сайта http://dejavu.sourceforge.net/
